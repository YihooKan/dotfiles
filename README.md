# dotfiles

My personal dotfiles for development environment.

## What's Inside

- **[zsh/](zsh/)** — Zsh configuration with Powerlevel10k theme and
  useful plugins
- **[nvim/](nvim/)** — Neovim configuration managed with lazy.nvim
- **[lazygit/](lazygit/)** — Lazygit configuration with Catppuccin Macchiato theme

## Quick Start

See individual folders for detailed setup instructions.

## Requirements

- Zsh (default shell)
- Oh My Zsh
- Neovim >= 0.9
- A [Nerd Font](https://www.nerdfonts.com/) (for Powerlevel10k and Neovim icons)
- [ripgrep](https://github.com/BurntSushi/ripgrep) (for Telescope live grep)
